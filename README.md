🌟 朋友关系模拟进化器
这是一个基于原生 HTML5 / CSS3 / JavaScript 开发的情感管理小工具。它通过色彩渐变条和动态位移，直观地展示你与朋友之间关系的亲疏远近。

✨ 核心功能
动态进化图谱：朋友的头像会根据“亲密度分数”在蓝（陌生人）到红（人生伴侣）的渐变条上实时滑动。

时间流逝系统：你可以设置“自动增长速度”，模拟时间推移下感情的自然升温。

事件调节尺 (Modal)：点击互动或矛盾按钮，通过滑动条自定义事件的影响力（±1 至 ±50 分）。

情感备忘录：支持记录每次互动的具体事件概述和发生时间。

历史日志查询：点击头像即可查看该好友完整的“成长日志”，追溯点滴往事。

本地化存档：数据通过浏览器 LocalStorage 存储，无需后端服务器，关闭网页后数据不会丢失。

个性化定制：支持本地上传头像图片，并自动进行 Base64 编码存储。

🛠️ 技术栈
前端：HTML5, CSS3 (Flexbox/Grid), JavaScript (Vanilla JS)

存储：浏览器本地存储 (Web Storage API)

图片处理：FileReader API (Base64 编码)

图标/头像：支持本地上传或自动生成 DiceBear 默认头像

🚀 快速开始
直接运行：
无需安装任何环境，双击 index.html 即可在浏览器中运行。

📖 使用指南
录入档案：在下方控制面板输入名字，上传一张精美的本地照片，设置初始分数和自动增长速度。

记录事件：点击头像下方的 + 互动 或 - 矛盾。在弹窗中拉动调节尺，输入当时的感受（如：“一起看了场很棒的电影”），确认后分值会立即更新。

查看往事：点击朋友的头像，系统会弹出该朋友的完整历史日志，按时间倒序排列。

管理档案：鼠标悬停在头像上，点击右上角的红色 × 即可永久删除该档案。

⚠️ 注意事项
由于数据存储在浏览器本地（LocalStorage），清理浏览器缓存会导致数据丢失。
